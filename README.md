# Maaş Hesaplama by Göksel

Türkiye için kapsamlı maaş hesaplama uygulaması.

## Özellikler

### Brütten Nete Maaş Hesaplama
- 2025 Türkiye Gelir Vergisi Dilimleri ile hesaplama
- SGK, işsizlik sigortası, gelir vergisi, damga vergisi hesaplamaları
- Medeni durum ve çocuk sayısına göre AGİ (Asgari Geçim İndirimi)
- Çeyreklik prim hesaplama (Ocak, Nisan, Temmuz, Ekim)
- İşveren maliyet hesaplamaları
- 12 aylık detaylı tablo
- Kümülatif vergi takibi
- Vergi dilimi dağılımı gösterimi

### Netten Brüte Maaş Hesaplama
- Hedef net maaşa göre gereken brüt maaş hesaplama
- Tüm parametreler ile otomatik hesaplama

### Tahmini Zam Hesaplama
- 3 farklı senaryo ile zam simülasyonu
- Ay ve oran bazlı hesaplamalar

### Kıdem, İhbar ve Yıllık İzin Hesaplama
- Kıdem tazminatı hesaplama (2025 tavanı ile)
- İhbar tazminatı hesaplama
- Yıllık izin ücreti hesaplama
- Kümülatif vergi hesaplamaları

## Kullanım

Ana sayfa: `index.html`
Tazminat hesaplama: `tazminat-hesaplama.html`
